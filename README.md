## Cài đặt

```bash
# Clone repo
git clone https://github.com/EGANY-Team/vct-8-optimize-reactjs-bundle
cd vct-8-optimize-reactjs-bundle

# Install packages
npm install

# Start dev server
npm start

# Build
npm run build
```
